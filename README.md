Top 10 Best Foods to Buy Right Now

A beginner-friendly HTML + CSS website listing the top 10 foods to buy, with images, descriptions, and some fun facts.

Features

- Lists the top 10 foods with descriptions and images
- Includes a table with cost, location, and estimated sales
- Clean and simple layout using HTML and CSS
- Responsive images with max-width  and height: auto
- Google Analytics tracking added

Technologies Used

- HTML
- CSS

How to View

1. Clone or download the repository
2. Open index.html in your browser
3. Enjoy browsing the top foods!
